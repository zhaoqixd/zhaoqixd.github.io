  • 为什么要阻止跨域? 有具体例子嘛?

    总的来说，就是跨域会导致安全问题，例如CSRF（跨站请求伪造）、iframe导致主页面崩溃等问题。

  • 如何实现跨域

    JSONP，Websocket，CORS(跨域资源访问，这个一般是后端配置的->延伸问题：简单请求和非简单请求)

  • 聊聊 HTTP 缓存

  • 聊聊 HTTPS

  • 聊聊 HTTP2 vs HTTP3

    HTTP2：二进制分帧、多路复用、服务端推送

    HTTP3：基于UDP的QUIC协议；更彻底的解决的队头阻塞的问题；保证切换网络的时候保持连接（基于UDP的QUIC协议，则可以内建与TCP中不同的连接标识方法，从而在网络完成切换之后，恢复之前与服务器的连接。）

  • 聊聊 CDN

    CDN 是一个内容分发网络，通过对源网站资源的缓存，利用本身多台位于不同地域、不同运营商的服务器，向用户提供就近访问的功能。也就是说，用户的请求并不是直接发送给源网站，而是发送给 CDN 服务器，由 CND 服务器将请求定位到最近的含有该资源的服务器上去请求。这样有利于提高网站的访问速度，同时通过这种方式也减轻了源服务器的访问压力。**资源放在离你请求最近的站点，使得快速访

  • History 改动怎么监听

    react 就使用useHistory监听，结束后卸载。

  • React key 有什么用

    diff算法中比较节点是否可以复用。（可以延申->react diff 算法）

  • React class 组件和 functional 组件区别

    类别 类组件 函数组件
    是否有状态 有 无
    是否有this值 有 无
    是否有生命周期 有 无

  • React 组件通信方式

    父子组件props传参，子调用父组件的回调；context传参；redux传参

  • React useMemo vs useCallback

    这个单独整理一篇博客吧，简单描述就是两个hooks是依赖项变换就会缓存结果的作用，memo是保存值，callback是保存函数。
